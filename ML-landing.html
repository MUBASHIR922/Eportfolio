Machine Learning

This module explores the fundamental principles and practical applications of Machine Learning, focusing on supervised and unsupervised learning techniques. Through regression, classification, clustering and model evaluation, the module develops the ability to build, interpret and assess predictive models using Python and industry-standard machine learning libraries. Practical activities emphasise data preparation, feature engineering, model evaluation and ethical considerations when deploying machine learning solutions for real-world business problems.

Learning Outcomes

I will learn:

  • Develop a critical understanding of the fundamental concepts, terminology and applications of machine learning.
  • Apply appropriate data preprocessing techniques including data cleaning, feature selection, encoding and scaling to prepare datasets for machine learning models.
  • Implement supervised learning algorithms such as Linear Regression, Logistic Regression and Decision Trees using Python.
  • Apply unsupervised learning techniques including clustering to identify patterns and relationships within datasets.
  • Evaluate machine learning models using appropriate performance metrics such as RMSE, MAE, Accuracy, Precision, Recall and R².
  • Critically assess the strengths, limitations and ethical implications of machine learning models in real-world decision-making.
  • Communicate analytical findings through clear visualisations, interpretation and evidence-based recommendations.
