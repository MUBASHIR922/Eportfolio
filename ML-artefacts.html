Unit 11 – Machine Learning Project Presentation

The final assessment involved developing and presenting a recommender system using the MovieLens 100K dataset. Our project compared Singular Value Decomposition (SVD) with Neural Collaborative Filtering (NCF) using Python, PyTorch and Scikit-learn. We evaluated both models using RMSE, MAE and NDCG while considering explainability through SHAP and LIME.
